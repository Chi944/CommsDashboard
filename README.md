# Global Commodities & Shipping Disruption Monitor

Single-page React dashboard (Vite + React + Tailwind + Recharts) that surfaces real-time-style
metrics on commodity prices and maritime chokepoint risk. All data is mocked.

## Run

```bash
npm install
npm run dev
```

## Tabs
- **Overview** — corridor risk index, hero stats
- **Prices** — 9 commodities with 30-day history
- **Routes** — 15 shipping lanes on a hand-coded SVG world map
- **Calculator** — voyage cost with war-risk multiplier
- **Intel** — curated mock news feed
